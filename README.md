# bebopCORE

Firmware para ESP32 destinado à criação de gamepads Bluetooth compatíveis com Nintendo Switch e dispositivos HID.
ESP32 firmware for creating Bluetooth gamepads compatible with Nintendo Switch and HID devices.

## 🕹️ Inputs

- D-Pad (Up, Down, Left, Right)
- Buttons A & B
- Start & Select
- L & R
- 4 Fn Keys
